Hello, World!

前言

  说了好久好久的个人网站终于上线了，第一篇正式的文章当然是用来记录这个对于我个人来说非常重要的时刻了。2019年3月29日，星期五，小雨，一个蓬头垢面，已经饿了好几顿的化工男终于准备正式开始他的博客之旅了。他现在心里有点激动，打字都有点手抖，因为他觉得未来可能会有很多人看到他现在正在写的这一篇文章，当然，也可能没有。但是不管未来有没有人看我的博客，至少现在的黄盼心里非常的高兴，因为他终于有了他梦寐以求的个人网站，一个可以将他的喜怒哀乐、所见所闻都装进去的网站。

原因

  至于我为什么执着于建立自己的网站，可能跟自己爱折腾有关系吧。从大二开始，自己就开始捣鼓各种各样的东西。在这个过程中，不可避免地会遇到各种各样的问题，我这个电脑小白只能求助于大佬们，在贴吧、博客、书籍、QQ 群里寻找大佬们留下的解决方案，而其中最多的就是大佬们的个人网站了。对于当时如小白般的我来说，拥有自己的网站是一件多么炫酷的事情。于是，大二的某一天，我开始自学前端三件套了。在这个过程中，逐渐了解到了 HTML、CSS 和 JavaScript 的基本语法，也照着书做了几个例子，但和大佬们做出来的效果实在是云泥之别。但心急吃不了热豆腐，我只好在网站上下载大佬们做好的网页，好好研究了。刚开始真的是到了每行代码都得百度的地步，但最后还是慢慢地熟悉了套路，做出了几个像样的网页。

小成就

  而我至今最得意的一次尝试是我大三那年花了两个星期的时间，给社团做了一个宣传网页，点击查看相关文章。虽然没有采用任何后端技术，但这是我第一次做那么大的项目，也是第一次将网页挂在互联网上。虽然现在看来，过程也挺简单的，下个模板，改一改，购买域名和空间，然后域名解析，最后 FTP 上传就完事了，但对于当年的我来说，这还是一个很大的挑战。还记得当时为了省钱和省事，空间申请的是国外的免费空间，所以那叫一个慢啊。不像现在，有钱了，舍得对自己投入了，花钱买了腾讯的云主机（在此要特别感谢中科研青能所支付了我网站一年的费用）。不过不管社团网页做得多么的烂，最后听到学弟学妹们的赞美，心里还是特别特别地高兴的。这也更加增强了我做个人网站的信心和渴望。

契机

  其实在这之前，我也曾经做过一个个人网页，不过很可惜的是，当时的我并没有完成它。我当时傻不拉几的把它给删除了。连同它一起删除的还有好多珍贵的东西呢。所以，现在想想，有时候云备份挺重要的。现在的我就可以通过这个平台，将我所有的东西传到我的网站主机里了，随时随地地查看了。

展望

  今天是我第一次正式写博客，所以思路有点乱。我打算过几天再告诉大家我的网站域名，要等文章比较多了来才行，所以这篇文章可能要很多天后才能和大家见面了。
  希望多年以后，我的外孙女依然能够在网上看到他外公几十年前写的东西，并为她有这么爱折腾的外公而感到自豪。
