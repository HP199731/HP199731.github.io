前言

  这个网页是博主大三在社团当会长时花了两个星期时间做的，主要是为了准备在“百团大战”上宣传社团。很简单的静态网页，但当时还是花了我很多精力，查阅了各种资料，最后效果还是不错。不过很可惜的是，我并没有坚持把这个网页做下去，所以网页上还有好多当时测试时留下来的痕迹。虽然采用了一些框架，但对于我这个非科班出生的人来说，做网页还是太痛苦了，可能这也是我没坚持做下去的原因之一吧。
